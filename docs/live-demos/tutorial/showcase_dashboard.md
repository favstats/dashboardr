# Showcase Dashboard

## Explore the Full Showcase

Ready to see all the features dashboardr has to offer?

The **Showcase Dashboard** demonstrates the complete power of the
dashboardr package, including:

- **Interactive Filters** - Page sidebars with checkbox, dropdown, and
  radio filters
- **Value Boxes** - Key metrics at a glance with custom styling
- **Input Controls** - Select, switch, checkbox, radio, and slider
  inputs
- **Content Blocks** - Callouts, cards, accordions, and more
- **Advanced Visualizations** - Multiple tabset groups with various
  chart types
