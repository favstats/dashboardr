# Minimal Tabset Theme

This dashboard demonstrates the **minimal** tabset theme with different
nesting levels.

## How to Use This Theme

``` r
create_dashboard(
  title = "My Dashboard",
  output_dir = "my_dashboard",
  tabset_theme = "minimal"
)
```

## Demo Pages

- **1 Level** - Simple tabs (Demographics, Regions)
- **2 Levels** - Nested tabs (Topic \> Subtopic)
- **3 Levels** - Deep nesting (Category \> Subcategory \> Detail)
