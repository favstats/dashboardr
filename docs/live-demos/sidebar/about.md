### About This Dashboard

This dashboard demonstrates the **multi-page sidebar** feature in
dashboardr.

Each page can have:

- Its own sidebar (or none)

- Different input types (checkbox, radio, select, slider)

- Left or right sidebar position

- Independent filtering for that pageâ€™s visualizations

- Multi-column grid layouts for checkboxes (using `columns = 2`)
