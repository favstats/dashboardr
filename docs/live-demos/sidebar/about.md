### About This Dashboard

This dashboard demonstrates the **multi-page sidebar** feature in
dashboardr.

Each page can have:

- Its own sidebar (or none)

- Different input types (checkbox, radio, select, slider)

- Left or right sidebar position

- Independent filtering for that pageâ€™s visualizations

- Multi-column grid layouts for checkboxes (using `columns = 2`)

------------------------------------------------------------------------

### Features Demonstrated

1.  **GDP Growth**: Left sidebar with checkbox filters in 2-column grid

2.  **Unemployment**: Left sidebar with multi-select dropdown

3.  **Combined**: Right sidebar with radio buttons + checkboxes

4.  **Slider Demo**: Left sidebar with year slider + checkboxes

------------------------------------------------------------------------

*Data is simulated for demonstration purposes.*
