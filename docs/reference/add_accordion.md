# Add collapsible accordion/details section

Add collapsible accordion/details section

## Usage

``` r
add_accordion(content, title, text, open = FALSE)
```

## Arguments

- content:

  A content_collection or viz_collection object

- title:

  Section title

- text:

  Section content

- open:

  Whether section starts open (default: FALSE)

## Value

Updated content_collection
