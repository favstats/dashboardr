# Apply legend position to a ggiraph/ggplot chart

Apply legend position to a ggiraph/ggplot chart

## Usage

``` r
.apply_legend_ggplot(p, legend_position, default_show = TRUE)
```

## Arguments

- p:

  ggplot object (before girafe())

- legend_position:

  Character: "top", "bottom", "left", "right", or "none"

- default_show:

  Logical: whether to show legend by default
