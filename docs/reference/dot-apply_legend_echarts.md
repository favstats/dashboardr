# Apply legend position to an echarts4r chart

Apply legend position to an echarts4r chart

## Usage

``` r
.apply_legend_echarts(e, legend_position, default_show = TRUE)
```

## Arguments

- e:

  echarts4r chart object

- legend_position:

  Character: "top", "bottom", "left", "right", or "none"

- default_show:

  Logical: whether to show legend by default (when legend_position is
  NULL)
