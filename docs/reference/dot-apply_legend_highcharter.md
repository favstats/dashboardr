# Apply legend position to a highcharter chart

Apply legend position to a highcharter chart

## Usage

``` r
.apply_legend_highcharter(hc, legend_position, default_show = TRUE)
```

## Arguments

- hc:

  highcharter chart object

- legend_position:

  Character: "top", "bottom", "left", "right", or "none"

- default_show:

  Logical: whether to show legend by default
