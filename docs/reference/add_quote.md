# Add a blockquote

Add a blockquote

## Usage

``` r
add_quote(
  content,
  quote,
  attribution = NULL,
  cite = NULL,
  tabgroup = NULL,
  show_when = NULL
)
```

## Arguments

- content:

  Content collection object

- quote:

  Quote text

- attribution:

  Optional attribution/source

- cite:

  Optional citation URL

- tabgroup:

  Optional tabgroup for organizing content (character vector for nested
  tabs)

- show_when:

  One-sided formula controlling conditional display based on input
  values.
