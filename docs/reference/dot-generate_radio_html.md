# Generate HTML for radio input

Generate HTML for radio input

## Usage

``` r
.generate_radio_html(
  input_id,
  label,
  filter_var,
  options,
  default_selected,
  width,
  align,
  inline,
  size = "md",
  help = NULL,
  disabled = FALSE,
  columns = NULL,
  stacked = FALSE,
  stacked_align = "center",
  group_align = "left",
  ncol = NULL,
  nrow = NULL
)
```
