# Add video

Add video

## Usage

``` r
add_video(
  content,
  src,
  caption = NULL,
  width = NULL,
  height = NULL,
  tabgroup = NULL,
  show_when = NULL
)
```

## Arguments

- content:

  A content_collection object

- src:

  Video source URL or path

- caption:

  Optional caption

- width:

  Optional width

- height:

  Optional height

- tabgroup:

  Optional tabgroup for organizing content (character vector for nested
  tabs)

- show_when:

  One-sided formula controlling conditional display based on input
  values.

## Value

Updated content_collection
