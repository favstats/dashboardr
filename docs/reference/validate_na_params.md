# Validate NA Handling Parameters

Validate NA Handling Parameters

## Usage

``` r
validate_na_params(include_na, na_label, param_name = "na_label")
```

## Arguments

- include_na:

  Logical. Whether to include NAs

- na_label:

  Character. Label for NA values

- param_name:

  Character. Name of parameter for error messages

## Value

Validated na_label
