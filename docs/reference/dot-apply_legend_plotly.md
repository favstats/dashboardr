# Apply legend position to a plotly chart

Apply legend position to a plotly chart

## Usage

``` r
.apply_legend_plotly(p, legend_position, default_show = TRUE)
```

## Arguments

- p:

  plotly chart object

- legend_position:

  Character: "top", "bottom", "left", "right", or "none"

- default_show:

  Logical: whether to show legend by default
